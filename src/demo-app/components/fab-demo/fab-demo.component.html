Floating Action Buttons

The MDC FAB component is a spec-aligned button component adhering to the Material Design button requirements.
Component
button[mdc-fab]
a[mdc-fab]
Properties Description
@Input() mini: boolean Makes the fab smaller (40 x 40 pixels).
@Input() plain: boolean Makes the FAB have a white background.
@Input() disableRipple: boolean
Child Directive
mdc-fab-icon 
<button mdc-fab
   [mini]="isMini"
   [plain]="isPlain"
   [disableRipple]="isRippleDisabled">
  <mdc-fab-icon material-icon>edit</mdc-fab-icon>
</button>
Using Font Awesome 
<button mdc-fab
  class="fa-icon">
  <mdc-fab-icon class="fa fa-keyboard-o"></mdc-fab-icon>
</button>
Text only 
<button mdc-fab>
  <mdc-fab-icon class="fa-icon">T</mdc-fab-icon>
</button>
